Differentiation

Tag Similarity Differentiation
hr & br

Break the line.

hr : break with the line.
br : break with the space.
ul & ol Show the list of something in separate line. ul : show the list with bullet point. (unordered list)
ol : show the list with the number. (ordered list)
select & checkbox & radio button

Show the list of something to choose.


select : create a drop-down list.
checkbox : show the list with the square box to tick, User can tick many box.
radio button : show the list with circle to tick, User can tick only one circle.
div & span Make the division of content. span is inline division.
script & style
Contain the script.

script is used to embed a client-side script (JavaScript).
style is used to define style information (CSS) for a document.

Input type

Input type Usability
text Defines a single-line text input field.
password Defines a password field.
button Defines a button for submitting form data to a form-handler.
reset Defines a reset button that will reset all form values to their default values.
radio Radio buttons let a user select ONLY ONE of a limited number of choices.
checkbox Checkboxes let a user select ZERO or MORE options of a limited number of choices.
button Defines a button, specify the event by user.
color Used for input fields that should contain a color.
date Used for input fields that should contain a date.
datetime-local Specifies a date and time input field, with no time zone.
email Used for input fields that should contain an e-mail address.
file Defines a file-select field and a "Browse" button for file uploads.
month Allows the user to select a month and year.
number Defines a numeric input field, you can also set restrictions on what numbers are accepted.
range Defines a control for entering a number whose exact value is not important (like a slider control).
search Used for search fields (a search field behaves like a regular text field).
tel Used for input fields that should contain a telephone number.
time Allows the user to select a time (no time zone).
url Used for input fields that should contain a URL address.
week Allows the user to select a week and year.
image Defines an image as a submit button. (The button symbol is image.)
hidden A hidden field let web developers include data that cannot be seen or modified by users when a form is submitted.
