Differentiation

Tag Similarity Differentiation
hr & br

Break the line.

hr : break with the line.
br : break with the space.
ul & ol Show the list of something in separate line. ul : show the list with bullet point. (unordered list)
ol : show the list with the number. (ordered list)
select & checkbox & radio button

Show the list of something to choose.


select : create a drop-down list.
checkbox : show the list with the square box to tick, User can tick many box.
radio button : show the list with circle to tick, User can tick only one circle.
div & span Make the division of content. span is inline division.
script & style
Contain the script.

script is used to embed a client-side script (JavaScript).
style is used to define style information (CSS) for a document.
